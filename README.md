# PrivateStorage 私人存储

PrivateStorage 是一个 Slimefun4 附属插件。
你需要安装 Slimefun4 才能让此插件运行。

该附属添加了两种新类型的箱子：私有(保险箱)与公共(箱子)。保险箱只能由放置该箱子的玩家打开或破坏，而公共箱子则遵循保护插件的规则。

该附属增加了不同木材类型的箱子、铁箱子、金箱子、钻石箱子以及防爆箱子等。
